# Thematic API

The thematic-api is our attempt to start bringing our API to the modern age. We'd like to go open-source with as many components as possible as well as making them using architectury to allow for NeoForge usage if desired.

The plans are to migrate the following in the future:
- Armor Rendering
- Armor Animations
- First person animations
- Ability System
  - Event based
  - Remove harsh conflicting mixins and create callbacks for them in it's place
- Armor Registration
  - Create a simpler setup to manage collections
